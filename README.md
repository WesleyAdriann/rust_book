### Cargo

Criar novo projeto
```sh
cargo new <name>
```

Buildar 
```sh
cargo build
```

Executar 
```sh
./target/debug/<name>
```

Buildar e executar
```sh
cargo run
```

Buildar projeto sem gerar executável para checar erros
```sh
cargo check
```

Buildar com otimizaçōes (prd)

```sh
cargo build --release
```

Atualizar dependencias (não ira ler o .lock)
```sh
cargo update
```

Abrir docs
```sh
cargo doc --open
```
