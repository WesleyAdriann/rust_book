### Cargo

Criar novo projeto
```sh
cargo new <name>
```

Buildar 
```sh
cargo build
```

Executar 
```sh
./target/debug/<name>
```

Buildar e executar
```sh
cargo run
```

Buildar projeto sem gerar executável para checar erros
```
cargo check
```

Buildar com otimizaçōes (prd)

```
cargo build --release
```
